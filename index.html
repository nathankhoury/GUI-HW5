One-Line Scrabble

What am I supposed to do?

The goal is to use the letters you are given to form words on the board. You can submit words to earn points based on the letter values and any special multipliers on the board. Try to get the highest score possible before running out of tiles!


Each tile displays a letter, along with its numerical value in the bottom right corner. The value of each tile is determined by how common the letter is in the English language. The blank tile is traditionally used as a wildcard letter, and as a result of its flexibility it has a point value of 0.


Some slots on the board have special multipliers that can increase the value of your word. The blue "Double Letter" slots will double the value of any tile placed on them, while the red "Double Word" slots will double the total value of the word if any tile is placed on them.

How am I supposed to do that?


You may use the Submit Word button to submit the word formed by the tiles you currently have arranged on the board. Word validity is not currently checked (so you can use that to your advantage if you wish ðŸ˜‰). Submitting a word will preserve any unused tiles, and promptly clear the board of all placed tiles. You will gain as many new tiles from the bag as it takes to fill your rack, and all the points that your word deserves!

The Return Tiles button allows you to return all tiles currently on the board back to the rack in the order that they were orginally distributed. This does not waste any tiles, nor does it give you any new tiles from the bag. It simply allows you to experiment with different word formations without inconvenience.

Using the Reset Game button will instantaneously reset the game state. Your score will be set back to 0, the bag will be refilled to 100 tiles, and both the board and rack will be cleared. This is useful if you want to start a fresh game without having to reload the page, but note that you will completely lose your current progress!

Lastly, the Pass button allows you to skip your current turn. This will clear all 7 tiles that you currently have available to use, and give you 7 brand new tiles from the bag. This is useful if you truly find yourself stuck and want to cut your losses. To reiterate, passing will permenently discard your current tiles, so use this option wisely.

Anything else to know?


It is important to understand how exactly tile placement is handled. If you board has no tiles on it, you may place a tile in any slot. However, once there is at least one tile on the board, any new tiles must be placed to the right of an existing tile.
